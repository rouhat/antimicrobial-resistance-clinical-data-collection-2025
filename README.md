This repository contains a curated dataset collected as part of a research project on antimicrobial resistance patterns in clinical samples.

The dataset includes anonymized patient and laboratory information such as:
- Sample type (urine, swab, etc.)
- Age and gender
- Organism identified
- Antibiotic susceptibility results (Sensitive / Intermediate / Resistant)

File:
- `data/Antimicrobial_Resistance_Data_Collection.xlsx`

## Dataset Description

Number of records: 183  
Number of variables: 73

Main variables include:
- Record Number
- Sample Collection Date
- Age (years)
- Gender
- Sample Type
- Growth Result
- Organism Identified
- Antibiotic susceptibility results for multiple agents
- Additional laboratory notes

The dataset is suitable for:
- AMR surveillance analysis
- Microbiology research projects
- Resistance trend visualization
- Teaching and academic purposes

## Ethical & Privacy Notes

All records are anonymized and contain:
- No personal identifiers
- No contact information
- No hospital / location identifiers

The dataset is shared strictly for academic and research use.
